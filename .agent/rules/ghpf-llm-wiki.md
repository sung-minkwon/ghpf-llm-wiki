# GHFP LLM Wiki Rule

When a task mentions GHFP, Obsidian wiki, LLM Wiki, papers, research notes, trading strategies, backtests, context packs, or agent memory:

1. Read `AGENTS.md`.
2. Prefer `wiki/` Markdown as the source of truth.
3. Use `_raw/` for intake and `swarmvault/` for sidecar artifacts.
4. Preserve user notes and make focused wiki edits.
5. Refresh graph/context outputs with `scripts/ghpf_wiki.py` when the wiki changes.
